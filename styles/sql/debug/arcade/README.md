# Arcade debug SQL

שאילתות סביב `public.arcade_device_sessions` (סשנים של משחקי הארקייד ב־device), עם קישור ל־`vault_balances` איפה שרלוונטי.  
התאמה לזרימות `start_freeplay_session`, `start_paid_session`, `finish_arcade_session` (ראה קבצי 13–15).

## לפני הרצה

- **`PUT-DEVICE-ID-HERE`** — החלף ב־`device_id` (טקסט).
- **`PUT-SESSION-ID-HERE`** — החלף ב־UUID של השורה ב־`arcade_device_sessions` (קבצים 18–19).

## בטוחים לבדיקה שגרתית (קריאה בלבד)

- **01–18** — סקירות, ספירות, תקועים ב־`started`, חיבור ל־vault, סכימה, הגדרות פונקציות (13–15), מלאי טבלאות MP אם קיימות (16), סיכום כמויות (17), שורה בודדת לפי session id (18).

## הרסניים — לא להריץ בלי גיבוי

- **19** — מוחק **סשן אחד** לפי UUID.
- **20** — מוחק **את כל** הסשנים של מכשיר אחד.

מומלץ לפני מחיקה: **01** או **02** / **18** כדי לוודא שזה המזהה הנכון.
